### Architecture Decisions Made

| ADR | Decision |
|-----|----------|
| ADR-001 | Flutter for mobile/desktop; native SwiftUI for tvOS |
| ADR-002 | Monorepo with scoped Dart packages |
| ADR-003 | Riverpod 2.x for state management |
| ADR-004 | Pure Dart Xtream API client (no native code) |
| ADR-005 | Firestore for cross-device sync + Hive for local |
| ADR-006 | RevenueCat + Stripe for subscriptions |
| ADR-007 | Cloudflare R2 + Stream for DVR storage |
| ADR-008 | Native SwiftUI for tvOS (not Flutter) |

### CI/CD Established

- GitHub Actions CI pipeline (analyze + test + iOS/Android/macOS builds)
- GitHub Actions Release pipeline (version tagging + draft releases)
- Branch protection on `main` (requires GitHub Pro)
- PR conventions + template
- CONTRIBUTING guide with commit message format
