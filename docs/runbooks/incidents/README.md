## How to Log an Incident

Create a new file: `docs/runbooks/incidents/YYYY-MM-DD-short-name.md`

Use the template in [INCIDENT_RESPONSE.md](../INCIDENT_RESPONSE.md#step-7--post-mortem).

After the post-mortem is written, add the incident to this table.
